## Technical Notes

### Typewriter Layout Shift Fix
Used CSS Grid with fixed row height instead of min-height to prevent subtitle from shifting during typewriter animation:
```css
.hero--left .hero-content {
  display: grid;
  grid-template-rows: 250px auto auto;
}
```

### Footer Reordering with CSS Grid
Used grid positioning to reorder footer sections without changing HTML:
```css
.footer-grid > div:nth-child(1) { grid-column: 1 / -1; grid-row: 1; }
.footer-grid > div:nth-child(2) { grid-column: 2; grid-row: 2; }
.footer-grid > div:nth-child(3) { grid-column: 1; grid-row: 2; }
.footer-grid > div:nth-child(4) { grid-column: 1 / -1; grid-row: 3; }
```

### CSS Variable Fallbacks
Replaced undefined `--radius-lg` variable with direct `12px` value for border-radius on images.

### Span Element Centering
Added `display: block` to `.overline` span since inline elements don't respond to text-align.
